## MAC
### 系统快捷键
先介绍一下键盘


| keyboard| describe|
| -- | -- |
| ⌘| command| 
| ⌃| control|
| ⌥| option|
| ⇧| shift |
| ⇪| caps lock|
    
   
窗口
   
| keyboard| describe|
| -- | -- |
| ⌃ + ↑/↓ |  调度 |
| ⌘ + tab | 切换窗口 |
| ⌘ + `   | 切换应用窗口 |
| ⌘ + H   |  隐藏窗口|
| ⌘ + M|最小窗口 | 
| ⌃ + ⌘ + 空格| 输入表情符号| 
| ⌃ + ⌘ + F| 全屏展示| 
 		
截图
| keyboard| describe|
| -- | -- |
| ⇧ + ⌘ + 3| 截取整个屏幕|
| ⇧ + ⌘ + 4| 截取区域|
| ⌃ + ⇧ + ⌘ + 3| 截取整个屏幕到剪切板|
| ⌃ + ⇧ + ⌘ + 4| 截取区域到剪切板|
